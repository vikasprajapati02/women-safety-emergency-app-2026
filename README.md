# Women Safety & Emergency Assistance Platform 🚨

A web-based Women Safety application designed to provide quick emergency support, secure login system, and safety tools for users. The project includes both frontend and backend components.

---

## 🔥 Features

- User Registration & Login System
- Emergency SOS Button for instant alert
- Quick emergency assistance interface
- Simple and responsive UI
- Backend support using Node.js (Express)

---

## 🛠️ Tech Stack

**Frontend:**
- HTML
- CSS
- JavaScript

**Backend:**
- Node.js
- Express.js

## 🚀 How to Run Locally

### 1. Clone the repository
```bash
git clone https://github.com/your-username/women-safety.git
```

### 2. Open project folder
```bash
cd women-safety/backend
```

### 3. Install dependencies
```bash
npm install
```

### 4. Run backend server
```bash
node server.js
```

### 5. Open frontend
Just open `index.html` in your browser.

---

## ⚠️ Important Notes

- GitHub Pages only runs frontend (HTML/CSS/JS)
- Backend (server.js) will not run on GitHub
- Use Node.js locally or deploy backend on platforms like Render / Railway
- Do NOT upload `.env` or sensitive files
